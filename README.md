# LEGALi — Stay On Guard, Stay Covered

Nigeria's premier online legal coverage platform — democratizing legal representation and making it accessible to all Nigerians.

## Quick Start

```bash
npm install && npm run dev
```

## Deploy to Cloudflare Pages

```bash
npm run build
CLOUDFLARE_API_TOKEN=your_token npx wrangler pages deploy dist --project-name=aura-academy --branch=main
```

## Demo Login

Visit `/login` → use demo buttons: **Customer**, **Lawyer**, or **Admin**
